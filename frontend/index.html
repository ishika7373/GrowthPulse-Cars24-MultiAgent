Architecture — Router + 4 Specialists + Supervisor

Every query enters the Router, an LLM classifier that emits structured JSON. Single-domain questions go to one specialist; cross-domain questions become MULTI and the Supervisor orchestrates 2+ specialists, then synthesises one prioritised plan.
